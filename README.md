# H7 Team Projects

这个仓库当前包含 3 个并列工程，分别对应战队不同比赛任务和测试用途。

## 工程说明

### 1. `DM4340`
- 用途：任务赛主工程
- 说明：当前仓库根目录就是 `DM4340` 主工程代码

### 2. `test dog h7`
- 用途：障碍赛主工程
- 说明：作为第二套正式工程并列放在仓库中

### 3. `test dog h7 bmi clean`
- 用途：状态补偿测试代码
- 说明：用于状态补偿相关功能的独立测试与验证

## 协作建议

- 日常开发先从 `master` 拉最新代码
- 每个人开发新功能时新建自己的分支
- 不同比赛工程尽量只修改各自目录，减少冲突
- 测试类代码优先放在对应测试工程中，不直接影响正式主工程
